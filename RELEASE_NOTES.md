# 发布说明

本文件用于记录每个对外版本的发布摘要，后续版本请按时间顺序继续追加。

## v0.1.0

发布日期：2026-04-08

### 新增

- 支持从本地 Codex 线程数据库查询线程记录
- 支持按线程 ID 前缀、标题前缀、首条用户消息前缀进行检索
- 支持在多个匹配结果中选择单个线程或全部线程导出
- 支持将线程导出为 Markdown
- 支持通过 `--db-path` 指定数据库路径
- 支持通过环境变量 `CODEX_STATE_DB` 指定数据库路径
- 支持通过 `--output-dir` 指定导出目录
- 支持命令行入口 `thread-exporter`

### 优化

- 导出结果按日期分层保存到 `output/YYYYMMDD/`
- 默认仅保留对话正文，减少噪音内容
- 忽略 `developer` 消息，连续 `assistant` 消息仅保留最后一条
- 增加基础测试覆盖
- 增加 GitHub Actions 持续集成

### 协作支持

- 提供 `CONTRIBUTING.md`
- 提供 `CODE_OF_CONDUCT.md`
- 提供 Issue 模板和 Pull Request 模板

### 说明

- 本项目面向本地使用场景，默认会尝试定位 `~/.codex/state_5.sqlite`
- 若你的环境使用不同路径，请通过参数或环境变量显式指定

## v0.1.1

发布日期：2026-04-08

### 优化

- 统一面向用户的文案为“对话”，避免“线程”表述带来的歧义
- 精简 README 首页，让结构更像正式开源项目
- 优化仓库 About 文案，突出检索、筛选和批量导出能力

### 说明

- `v0.1.0` 作为首个开源版本保留不变
- `v0.1.1` 仅包含文案与展示层面的整理，不影响核心导出逻辑
