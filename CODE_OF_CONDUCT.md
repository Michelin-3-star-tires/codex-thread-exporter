# 社区行为准则

本项目欢迎所有人的参与。为了让协作保持高效、友善和专业，所有参与者都应遵守以下原则：

## 我们承诺

- 以尊重、耐心和建设性的方式交流
- 关注问题本身，不攻击个人
- 在讨论分歧时尽量基于事实、需求和代码进行沟通
- 对新手问题保持包容，尽量清楚说明背景和原因
- 在 Issue、PR、评论和线下/线上讨论中保持礼貌

## 我们不接受

- 人身攻击、嘲讽、歧视或侮辱性言论
- 恶意刷屏、骚扰、故意破坏协作秩序
- 公开泄露他人隐私、密钥或敏感信息
- 以“开玩笑”为名的冒犯性内容

## 发现问题怎么办

如果你遇到不友善行为，或者认为某次互动违反了本准则，请直接通过 Issue 或你认为合适的私下渠道反馈。

## 处理方式

维护者会根据具体情况采取适当措施，包括但不限于：

- 提醒并要求修改不当言论
- 暂停或关闭相关讨论
- 限制协作权限
- 在必要时移除违规内容

## 适用范围

本准则适用于项目仓库内的所有公开协作场景，包括：

- Issue
- Pull Request
- 代码评审
- 文档讨论
- 相关社区交流

## 说明

本文件用于维护健康的协作环境，不替代任何法律建议或平台政策。
